# Lovable Prompt Pack — Quattro

Paste these into Lovable **after** adding DESIGN_DNA.md to Knowledge.

## 1. Establish the system
```
Use the Design DNA in my Knowledge as the strict visual system for this project. It is a dark theme built on #000000 backgrounds, #ffffff text, Inter typography, and #0099ff as the accent. Apply it to everything you build.
```

## 2. Build a landing page
```
Build a landing page in the Quattro design system from my Knowledge: hero with headline + subtext + primary CTA, a 3-card feature section, social proof strip, pricing, FAQ, and footer. Match the spacing rhythm and only use the accent color for CTAs.
```

## 3. Add a page consistent with the system
```
Add an About page that matches the exact same Design DNA — same fonts, colors, spacing grid, radius, and section rhythm as the rest of the site.
```

## 4. Enforce the tokens
```
Configure tailwind.config.js using the token snippet I provide, then use only those theme tokens (bg-background, text-foreground, bg-primary, etc.) — no arbitrary hex values.
```
